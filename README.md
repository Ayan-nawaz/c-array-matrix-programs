# C Array Matrix Programs

A collection of C programs demonstrating fundamental array and matrix operations. This repository contains beginner-friendly programs for practicing common array and 2D matrix concepts in C.

## Programs Included

1. **basic-2D.c**
   - Demonstrates the basics of 2D arrays, including input and output of matrix elements.

2. **dublicate-elements.c**
   - Finds and displays duplicate elements in an array.

3. **uni-elements.c**
   - Identifies and prints unique elements present in an array.

4. **store-10-five-by-five.c**
   - Stores and displays elements in a 5×5 matrix.

5. **odd-matrix.c**
   - Finds and displays odd elements from a matrix.

## Topics Covered

- Matrix Operations

- Unique Elements
